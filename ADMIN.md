# Админка АСКАО

Админка управляет четырьмя разделами:

- Новости
- События
- Видео
- Комитеты

## Запуск

Откройте два терминала в папке проекта.

В первом терминале запустите сайт:

```powershell
npm.cmd run dev
```

Во втором терминале запустите локальный backend админки:

```powershell
npm.cmd run cms
```

После запуска откройте:

```text
http://localhost:8080/admin/
```

## Публикация материала

1. Выберите нужный раздел в левом меню.
2. Нажмите кнопку создания новой записи.
3. Заполните обязательные поля.
4. Нажмите `Publish` или `Сохранить`.
5. Проверьте материал на сайте.
6. Соберите production-версию:

```powershell
npm.cmd run build
```

7. Загрузите содержимое папки `dist` в папку сайта на хостинге.

## Slug

Slug используется как имя файла и идентификатор записи.

Допустимы:

- строчные латинские буквы;
- цифры;
- дефис.

Пример:

```text
forum-atomstroy-2026
```

Slug существующей новости также используется в её адресе:

```text
/news/forum-atomstroy-2026
```

## Видео Rutube

В поле `Ссылка Rutube` вставляйте обычную ссылку:

```text
https://rutube.ru/video/d3c0ed16289daf74136c7baf68a60aa2/
```

Сайт автоматически преобразует её во встроенный плеер.

## Важно

Локальная админка работает только при одновременно запущенных командах `dev` и `cms`.
После изменения контента production-сайт нужно пересобрать и повторно загрузить на хостинг.
